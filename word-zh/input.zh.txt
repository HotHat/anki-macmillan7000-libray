input 
BrE ˈɪnpʊt, AmE ˈɪnˌpʊt 

A.  noun 
① Countable(of money, resources) 投入 tóurù (of energy) 输入 shūrù the input of [something] in or into [something]; 对某事物投入某资源 
② Countable(contribution) 贡献 gòngxiàn 
③ CountablePhysics, Electronics (energy) 输入能量 shūrù néngliàng the input is a low-frequency signal 输入的是低频信号 
④ CountableElectronics (device, port) 输入端 shūrùduān the input is located by the keyboard 输入端口在键盘上 
⑤ UncountableComputing 输入 shūrù the input of new information and data 新信息与数据的输入 
B.  transitive verb (present participle -tt-; past tense, past participle input or inputted) 输入 shūrù to input data into a computer 把数据输入计算机
