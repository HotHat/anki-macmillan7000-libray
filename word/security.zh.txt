security 
BrE sɪˈkjʊərɪti, sɪˈkjɔːrɪti, AmE səˈkjʊrədi 
noun 
① Uncountable(of person, state, investment) 安全 ānquán security from or against [something]; 免遭某物 security of employment 就业安全 
② Uncountable(procedures, measures) 安全措施 ānquán cuòshī security is very strict at the site 这个工地的安保措施非常严格 a security barrier/camera 安全屏障/监控摄像头 
③ Uncountableplus singular or plural verb (department) 保安部门 bǎo'ān bùmén call security! 去叫保安！ 
④ Uncountable and countable(guarantee) 担保 dānbǎo to take/leave [something] as security 接受某物/把某物作为抵押 to stand security for [somebody] 为某人担保 
⑤ Countable and uncountable(safeguard) 保障 bǎozhàng security against [something]; 免受某物之苦的保障 
⑥ Countableusually plural Finance 证券 zhèngquàn security company/market/trading 证券公司/市场/交易
